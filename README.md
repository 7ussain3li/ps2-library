# PS2 Library (موسوعة بلايستيشن 2)

A comprehensive tribute website dedicated to the legendary PlayStation 2 console, celebrating its cultural and technological impact on the gaming world.

## Summary

This project showcases the PlayStation 2 (PS2), Sony's groundbreaking gaming console released in 2000. With over 155 million units sold and a library of more than 3,800 games, the PS2 revolutionized gaming and home entertainment. The website provides historical context, technical specifications, and highlights of its most iconic titles.

## Features

- **Home Page**: An engaging introduction to the PS2's history, sales success, and innovative hardware components like the Emotion Engine and Graphics Synthesizer.
- **Games Library**: A curated list of 10 top PS2 games, featuring dynamic "glass shine" hover effects and beautifully staggered fade-in animations.
- **Contact Page**: An interactive form to suggest new games. It uses custom JavaScript to securely encode Arabic inputs (`UTF-8`) for seamless email transmission without character corruption.
- **Cyberpunk/Neon Aesthetics**: A striking dark theme with animated glowing gradients, neon titles, and a custom-styled scrollbar tailored to the gaming vibe.
- **Smooth User Experience**: Implemented smooth scrolling and responsive design for an immersive experience.
- **Arabic Layout (RTL)**: Fully optimized Right-to-Left interface utilizing the elegant Google 'Tajawal' font.

## Technologies Used

- **HTML5**: Semantic markup and structure.
- **CSS3**: Advanced styling, Flexbox, CSS Grid, `@keyframes` animations, glowing drop-shadows, and custom scrollbars.
- **JavaScript**: Form event handling and precise URI text encoding (`encodeURIComponent`).

## Development

This website was developed using HTML, CSS, and AI Vibe Coding tools to create an informative and visually appealing experience for PS2 enthusiasts.

## Usage

To view the website, simply open the HTML files in any modern web browser. No additional setup or server is required.

## Author

Developed by Hussain as an academic project.
